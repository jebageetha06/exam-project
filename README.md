# Exam Project 🎓

This is a full-stack exam application built with:

- **Frontend:** React + Vite  
- **Backend:** FastAPI (Python)  
- **Database:** MySQL  

## Features
- User Registration & Login 🔑  
- Start Exam with Timer ⏱️  
- Answer Questions and Submit ✅  
- View Results 📊
